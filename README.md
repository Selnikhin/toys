Bags and Toys

This is my newFlutter project.

Написан на Flutter 3.3.8

Приложение запускается на ios выше 12.0, а также на android выше 9.0

В приложении применялись технологии: 
 1. flutter_bloc
 2. Navigator
 3. Provaider

Использовались такие зависимости как:
1. carousel_slider: ^4.2.1
2. flutter_bloc: ^8.1.2
3. equatable: ^2.0.5
4. favorite_button: ^0.0.4
5. flutter_slidable: ^2.0.0

В дальнейшем планируется подключение FireBase
таких зависимостей как
1. firebase_auth: ^4.3.0
2. firebase_core: ^2.8.0
3. cloud_firestore: ^4.4.5 )

В РАЗРАБОТКЕ:

1. Перевести локальные ресурсы в Firebase
2. Сделать аунтификацию через Firebase
3. Сделать страницу "Оформления заказов" ( Возможно с подключением оплаты картами) 
4. Загрузить приложение в Apple store и\или в Play Market
